# AI 코디 추천 서비스 PRD

## 1. 개요
본 문서는 퍼스널 컬러와 사용자의 선택 정보를 기반으로 AI가 의상 코디 및 향기(플르부아 핸드크림)를 추천하는 서비스의 제품 요구사항 정의서(PRD)이다. 실제 개발에 즉시 활용 가능한 수준으로 기능, 화면, 기술 스택, 데이터 구조를 명확히 정의한다.

---

## 2. 목표
- 사용자의 퍼스널 컬러 및 부분별 의상 입력을 기반으로 AI 코디 추천 제공
- 미입력 영역에 대해 AI 자동 보완 추천
- 의상 + 향기 + 액세서리 + 메시지까지 포함한 완성형 코디 제안
- 단일 페이지 기반의 직관적인 UX 제공

---

## 3. 주요 기능

### 3.1 사용자 기본 정보 설정
**필수 입력**
- 성별: 남성 / 여성

**퍼스널 컬러**
- 메인 타입 (필수)
  - 겨울 쿨톤
  - 여름 쿨톤
  - 가을 웜톤
  - 봄 웜톤
- 세부 타입 (선택, 직접 입력)
  - 예: 겨울 딥, 여름 뮤트 등

---

### 3.2 부위별 의상 입력
사용자는 아래 부위별로 의상 정보를 선택적으로 입력할 수 있다.

| 부위 | 입력 항목 |
|---|---|
| 아우터 | 종류, 색상 |
| 상의 탑 | 종류, 색상 |
| 상의 미드 | 종류, 색상 |
| 상의 이너 | 종류, 색상 |
| 하의 | 종류, 색상 |
| 양말 | 종류, 색상 |
| 신발 | 종류, 색상 |

예시:
- 부위: 아우터  
- 종류: 데님 트러커 재킷  
- 색상: 애쉬 다크 그레이

---

### 3.3 AI 코디 추천
- 사용자가 입력한 정보(성별, 퍼스널 컬러, 일부 의상)를 기반으로 AI 추천 수행
- 입력되지 않은 부위는 AI가 자동으로 보완하여 조합 제안
- 추천 결과에는 다음이 포함됨:
  1. 전체 의상 코디
  2. 코디에 어울리는 플르부아(PLEUVOIR) 핸드크림 추천 (향수 대용)
  3. 추가 악세서리/소품 제안
  4. 오늘의 날씨를 고려한 스타일 인사이트
  5. 최종 코디가 전달하는 메시지를 문장으로 표현

---

## 4. 화면 구성 (1 Page)

### 4.1 메인 랜딩 페이지 구성
1. 서비스 로고 및 이름 영역
2. 서비스 사용 방법 소개 영역
3. 성별 입력 영역 (필수)
4. 퍼스널 컬러 입력 영역
   - 메인 타입 선택 (필수)
   - 세부 타입 직접 입력 (선택)
5. 부위별 의상 입력 영역
6. AI 추천 결과 출력 영역
